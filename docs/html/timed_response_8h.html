◆ timedResponse

unsigned char timedResponse
extern

TimedResponse handles the sending of CBUS responses at a slower rate to allow receivers of these messages to process them.

Two global variables are used to control the process:

  • timedResponse determines which set of responses is currently being sent.
  • timedResonseStep determines how far through the responses we're currently at

Call initTimedResponse() before other TimedResponse processing. This will set the current response set to be None. Call doTimedResponse() on a regular basis which will do the work needed for the next step and increments the step counter.

To start send a set of timed responses set the type of response required and set the step to 0. For example to send responses to NERD do: timedResponse = TIMED_RESPONSE_NERD; timedResponseStep = 0;
